# genaihub_startercode

Starter examples and utilities for working with GenAI Hub models and APIs.

## Where To Start

The main project documentation lives in:

- `GenAI Hub Starter/README.md`

That README includes:
- setup and environment variable instructions
- runnable Python examples for chat and embeddings
- available model names reference file (`GenAI Hub Starter/all_model.txt`)
- SSL troubleshooting for corporate/proxy environments
- OS-specific SSL fix scripts:
	- `GenAI Hub Starter/Cert-Solution-Windows.ps1`
	- `GenAI Hub Starter/fix_ssl_linux.sh`
	- `GenAI Hub Starter/fix_ssl_macos.sh`

## Models List

To see available model names you can use in your `.env` file, check:

- `GenAI Hub Starter/all_model.txt`

## Quick SSL Fix Reference

- **Windows:** Run `GenAI Hub Starter/Cert-Solution-Windows.ps1` in PowerShell
- **Linux:** Run `GenAI Hub Starter/fix_ssl_linux.sh`
- **macOS:** Run `GenAI Hub Starter/fix_ssl_macos.sh`

See the full instructions in `GenAI Hub Starter/README.md`.
